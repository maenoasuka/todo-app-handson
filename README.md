# Todo App Handson

タスク管理アプリケーションのハンズオンプロジェクトです。

## 概要

シンプルなタスク管理アプリケーションです。ブラウザだけで動作し、サーバーサイドの環境は不要です。
タスクの追加・編集・削除・完了管理ができ、データはブラウザのローカルストレージに保存されます。

## 技術スタック

| 技術 | 用途 |
|------|------|
| HTML | アプリケーションの構造・マークアップ |
| CSS | スタイリング・レイアウト |
| JavaScript | アプリケーションのロジック・DOM操作 |
| LocalStorage | タスクデータの永続化（ブラウザ内保存） |

## 動作環境

- モダンブラウザ（Chrome / Firefox / Safari / Edge）
- サーバー不要（HTMLファイルをブラウザで直接開いて動作）

## 使い方

1. `index.html` をブラウザで開く
2. テキストボックスにタスクを入力して追加
3. タスクをクリックして完了/未完了を切り替え
4. 削除ボタンでタスクを削除

## ファイル構成

```
todo-app-handson/
├── index.html   # メインHTML
├── style.css    # スタイルシート
├── app.js       # アプリケーションロジック
└── README.md    # このファイル
```
